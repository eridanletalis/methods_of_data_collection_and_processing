HeadHunter and Superjob vacancy parser
=====================
Вторая домашняя по курсу **Методы сбора и обработки данных из сети Интернет** 
-----------------------------------

При запуске спрашивает, какую строку искать на обоих агрегаторах и количество просматриваемых страниц каждого агрегатора.

В самой программе создаётся Pandas dataFrame, который по окончанию работы сохраняется в виде csv (UTF-16, разделитель ";").
